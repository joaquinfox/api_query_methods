    Below is a basic API call made using the fetch API. The request is made from a JavaScript file imported into a vanilla html file, with no framework or library.

            fetch(`https://randomuser.me/api/`)
              .then((response) => response.json())
              .then((response) => {
                const userData = response.results[0];
                const userName = userData.name;
                const userLocation = userData.location;
                const userPicture = userData.picture;
                h4El.append(`
                ${userName.title}
                ${userName.first} 
                ${userName.last}`);
                imageEl.src = userPicture.medium;
                locationSpan.append(`
                ${userLocation.city}, 
                ${userLocation.country}`);
              });
